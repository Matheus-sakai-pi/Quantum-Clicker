# Quantum Clicker 🚀

Quantum Clicker é um jogo incremental criado com HTML, CSS e JavaScript.

O jogador acumula Poeira Cósmica clicando, compra upgrades, desbloqueia ranks e pode realizar o Big Bang para reiniciar o progresso e escolher bônus especiais.

## Funcionalidades

* Sistema de cliques
* Upgrades de poder
* Produção automática por segundo
* Sistema de ranks
* Prestígio (Big Bang)
* Escolha de bônus
* Interface estilizada com CSS

## Tecnologias usadas

* HTML
* CSS
* JavaScript

## Objetivo

Este projeto foi desenvolvido como prática para aprender lógica de programação e manipulação do DOM com JavaScript.
